BookaRoom

This is a MERN project. This project is like airbnb where users can login and put up a house for rental or can book a room.
Features:
1. Users can signup and login(Authorization & Authentication)
2. Validations
3. User can create edit delete it's rental.
4. Use of google api to show the rental house on map.
5. Users can view other rentals and can book a rental if the rental is available for booking.
6. Separate section to view the rentals and the bookings of the user.
7. User can't book the place for the booked days.
8. User can search the house using the city where they want to find
9. Modals and beautiful GUI.
